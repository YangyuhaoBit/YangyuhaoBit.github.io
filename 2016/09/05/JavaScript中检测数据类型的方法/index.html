JavaScript中检测数据类型的方法

发表于 2016-09-05

引言

检测数据类型的方法不外乎4种，即

  1. typeof
  2. instanceof
  3. constructor
  4. Object.prototype.toString

4种方法各有优缺点，本文将分别介绍4中方法的用法即各自适用的场景。
